## ğŸš€ Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/SreeCharanManne/StandardPlantShopee.git
cd StandardPlantShopee
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open http://localhost:5173 in your browser

### Build for Production

```bash
npm run build
```

## ğŸ“§ Contact Form Setup

The contact form uses [Web3Forms](https://web3forms.com/) to send emails without a backend server.

To enable the contact form:

1. Go to https://web3forms.com
2. Enter your email address to get a free access key
3. Open `src/components/Contact.tsx`
4. Replace `'YOUR_WEB3FORMS_ACCESS_KEY'` with your actual access key
